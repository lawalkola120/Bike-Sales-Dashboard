# Bike-Sales-Dashboard
This repo contains a bike sales analysis project done on Microsoft Excel

## Project Overview

The Bike Sales Analysis Dashboard analyzes customer demographics and purchasing behavior to understand the factors that influence bike purchases. The dashboard was created using Microsoft Excel with Pivot Tables and charts to transform raw data into meaningful business insights.

## Tools Used
	-	Microsoft Excel
	-	Pivot Tables
	-	Pivot Charts
	-	Data Filtering (Slicers)

  ## Dashboard Features
  - Key Performance Indicators (KPIs)
	-	Total Customers: 1,026
	-Total Bikes Purchased: 495
	-	Average Buyer Income: 56.2K
	-	Cars Owned by Bike Buyers: 600
